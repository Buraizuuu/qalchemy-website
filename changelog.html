What's new in QAlchemy

Release notes and version history.

v1.3 Latest 2026
  • Added the Test Execution Workspace: run generated suites with Pass/Fail/Blocked/No Run tracking, environment and build details, and full execution history.
  • Added the Evidence Vault: organize screenshots, recordings, and documents into cases with pinning and preview support.
  • Merged the Media Compressor into the QA Toolbox for local image and video compression.
v1.2 2026
  • Added favorite/pin support to the QA Toolbox tool list, so frequently used utilities stay at the top.
  • Expanded the QA Toolbox with additional offline utilities.
v1.1 2026
  • Refreshed the in-app About screen and product information.
  • General reliability and polish improvements across the desktop app.
v1.0 2026
  • Initial release: AI-powered test case generation from user stories and acceptance criteria.
  • Test Suites workspace, multi-format export, and the first QA Toolbox utilities.
  • Support for Anthropic, OpenAI, Gemini, and Ollama as AI providers.
